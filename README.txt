# Nmap Network Scanning Project

## Project Title
Nmap Network Scan on Public DNS (8.8.8.8)

## Student Information
- **Name:** Sathvika Bogam  
- **Internship Role:** Network Security Intern  
- **Tool Used:** Nmap v7.97  
- **System:** Apple M1 (macOS)

## Objective
To perform a basic network scan using Nmap on Google's public DNS server and understand the use of Nmap in identifying open ports and services.

## Setup Instructions

1. **Install Nmap:**
   ```bash
   brew install nmap
Verify Installation:
/opt/homebrew/bin/nmap --version
Run the Scan:
sudo nmap 8.8.8.8
Save the Results:
Redirect output to a file:
sudo nmap 8.8.8.8 > scan_result.txt
Output Description

The scan was successfully completed on IP address 8.8.8.8.
The results show the open ports and services running on the target IP.
The output is saved in scan_result.txt.
Files Included

README.md: Documentation of the project.
scan_result.txt: Nmap scan output.
Conclusion

This project demonstrates the use of Nmap for basic network scanning and security auditing as part of an internship assignment.
