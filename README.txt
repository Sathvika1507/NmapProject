# Nmap Network Scan - Internship Project

## Student: Sathvika Bogam
## Tool Used: Nmap 7.97
## Platform: macOS (Apple Silicon)
## Target: 8.8.8.8 (Google DNS)

### Steps:
1. Installed Nmap via Homebrew
2. Verified installation: /opt/homebrew/bin/nmap
3. Ran scan: `sudo nmap 8.8.8.8`
4. Output saved in scan_result.txt

This project was created as part of my internship to demonstrate the use of network scanning tools like Nmap.
